HTMLda dasturlash

HTML dasturlash tilini mahsus ta'lim tizimi asosida o'rganishingiz va egallagan bilimingizni bizning test dasturimiz orqali doyimo tekshirib borishingiz mumkin.


CSS dasturlash tili

CSS dasturlash tilini mukammal o'quv qo'llanmasi asosida o'rganib barcha imkoniyatlaridan foydalanishingiz mumkin.Css dasturlash tilining barcha muhim jihatlari qamrab olingan o'quv dasturi sizni professionallikka yetaklaydi.


Java Script dasturlash tili

JavaScript dasturlash tili mashxur tillardan hisoblanadi.Uni o'rganish sizga keng imkoniyatlar eshigini ochadi.Bilimlaringizni haqiqiy loyihalarda sinab foydali web dasturlar yaratishingiz mumkin.
