Biz haqimizda

01 · HTML

HTMLda dasturlash

HTML dasturlash tilini maxsus ta'lim tizimi asosida o'rganishingiz va egallagan bilimingizni bizning test dasturimiz orqali doimo tekshirib borishingiz mumkin.

02 · CSS

CSS dasturlash tili

CSS dasturlash tilini mukammal o'quv qo'llanmasi asosida o'rganib, barcha imkoniyatlaridan foydalanishingiz mumkin. CSS dasturlash tilining barcha muhim jihatlarini qamrab olgan o'quv dasturi sizni professionallikka yetaklaydi.

03 · JavaScript

Java Script dasturlash tili

JavaScript dasturlash tili mashhur tillardan hisoblanadi. Uni o'rganish sizga keng imkoniyatlar eshigini ochadi. Bilimlaringizni haqiqiy loyihalarda sinab, foydali web dasturlar yaratishingiz mumkin.
